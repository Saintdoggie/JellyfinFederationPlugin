Federation

Watch libraries from other Jellyfin servers as if they were your own. Connect a server, pick what to share, save — that's it.

1 · Connect your servers

Add each remote Jellyfin server with its address and an API key (Dashboard → API Keys on that server).

New servers are tested automatically when added. Defaults work out of the box — streaming mode and other dials live under each server's "Connection details".

2 · Choose what to share

Each library you check appears on this server as a virtual library. Libraries with the same name on different servers are merged into one — duplicates are combined automatically.
